## 功能点
### 话务条操作
 * 拨号
 * 转接
 * 邀请三方通话
 * 呼叫保持与解除呼叫保持
 * 挂机
 * 回拨与预约
### 相关设置
 * 角色权限
 * 组织机构
 * 节假日管理
 * 黑/白名单管理
 * 工单类型设置
### 坐席监控及报表统计
 * 技能组监控
 * 坐席地图
 * 报表统计
